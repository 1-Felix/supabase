## How we use Supabase

アンドエーアイでは自社アプリとして女性向けのコスメ管理アプリである「フーコスメ」を開発・運営しています。元々は Firebase を利用していましたが、アプリの機能が拡張されるにつれ複雑なクエリを処理する必要が出てきたことや、リレーションが多く発生し、NoSQL に限界を感じたため Supabase への移行を決定しました。
そのため Firebase から Supabase へのマイグレーションには実績があります。
Supabase は開発効率の良さやスケーラビリティの高さが採用理由です。

## Our Ideal Supabase Project

Supabase を採用する理由はその開発効率の高さやスケーラビリティです。顧客がバックエンドの開発コストを抑えてフロントエンドの開発や事業成長に集中してリソース投下できることが最も大きなメリットだと考えています。
そのため新規事業やスタートアップなどのプロジェクトへは Supabase を積極的におすすめしたいと考えています。
